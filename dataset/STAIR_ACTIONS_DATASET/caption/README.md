## ファイル

- caption.csv
    - キャプションが記載されたファイル。一行目がヘッダーになっています。

## 利用規約

ユーザは本データセットをダウンロードしたことにより以下の利用規約に同意するものとします

- 本データセットの用途を人工知能研究に限ること

- 本データセットの利用によって生じるいかなる損害に対しても損害賠償請求をしないこと
